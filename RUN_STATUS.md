
Backend:

```text

```

Web app:

```text
http://localhost:5173/
```

The backend is actively syncing FTC API data. The current log shows authenticated FTC API requests and progress through the 2020 match load.

To view logs:

```bash
tail -f screenlog.0
```

To attach to a running session:

```bash
screen -r ftc-server-dev
```

Detach from a screen session with `Ctrl-a` then `d`.

To stop everything:

```bash
screen -S ftc-common -X quit
screen -S ftc-server-watch -X quit
screen -S ftc-server-dev -X quit
screen -S ftc-web-dev -X quit
```

## Manual Run Needed

Run the long-lived commands in separate terminals:

```bash
npm run common:watch
npm run server:watch
npm run server:dev
```

After the server is running and syncing:

```bash
npm run web:gen
npm run web:dev
```

The first server data sync can take a long time.
